## 🚀 这框架解决什么问题？

**A股散户的三大死穴：**

| ❌ 死穴 | 典型表现 | 框架怎么破 |
|---------|---------|-----------|
| **信息差** | 追热点时已经涨完了，一买就套 | **BOM拆解** — 在热点爆发前就锁定关键环节的龙头 |
| **拿不住** | 好票跌10%就慌，卖飞后翻倍 | **三高评分+辩论验证** — 有数据撑腰，下跌敢加仓 |
| **全凭感觉** | 这家公司"听起来不错"就买，不知道为什么涨跌 | **政策漏斗+逻辑退出** — 每一笔买入卖出都有明确依据 |

**一句话：** 不是教你预测明天涨跌，是给你一套**可复制、可复盘、可迭代**的选股操作系统。

## 💡 这个框架适合谁？

**适合你，如果：**
- 资金量1万-100万，不是机构也不是游资
- 不想天天盯盘，每周花1-2小时研究
- 相信"国产替代"是未来5-10年最大的投资主线
- 愿意按照规则执行，而不是凭感觉交易

**不适合你，如果：**
- 追求短期翻倍，每天都要交易
- 只信内幕消息或者只想抄作业
- 没有耐心读完一份框架文档
